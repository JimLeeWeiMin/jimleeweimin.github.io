Second Scholar

Second Scholar

Sep 2017 - Apr 2018

Project description

Second Scholar is an isometric puzzle game where you play as a scholar on a quest to save the jade emperor.

This project is completed in a span of two semesters in Digipen. The first semester, GAM200, was used to construct a game engine on from scratch and the second semester, GAM250, is to put the game into the engine.

It is done using Visual Studio with Win32 API as its base and OpenGL as its Graphics Library.

Awarded

Claude Comair Grand Prize for Game of the Year - 2nd Place, 2018

Work on 2D graphic, 2D particles and other miscellaneous.
