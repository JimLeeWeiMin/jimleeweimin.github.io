Project description

Tagger is a multiplayer game, a third-person action game where two teams compete against each other in a game of tag.

This project is completed in a span of a semester as part of our final year project at Temasek Polytechnic in a team of five.

It is done using Unity.

Awarded

Selected to be part of the showcase in Temasek Polytechnic IT Project Show 2014

Role - Programmer

Work on network architecture and scripting in C#
