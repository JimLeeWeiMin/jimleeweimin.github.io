Project description

Developed an engine to support object recognition in real-time or a given video at SIT Singapore working on an embedded system, Jetson TX2 on Unix OS

With little experience with an embedded system, python, Unix, and live detection, I have developed an interface that allows the user to draw areas where the user would like to count if a person decides to walk over the area or wait in an area.

When the user is satisfied, the user can run the program to exit out of the editor and into the live detection where it will continue to exist till the user decide to exit out of the program and back into the editor.
