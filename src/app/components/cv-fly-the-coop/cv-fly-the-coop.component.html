Fly the Coop

Fly the Coop

Sep 2018 - Mar 2019

Project description

Fly The Coop is a third-person adventure-comedy game where you play as a chicken who has been outcasted and hunted by none other than your own kind. The game takes the player on a journey to find out why he is hunted and the eventual truth behind it.

This project is completed in a span of two semesters in Digipen. The first semester, GAM300, was used to construct a game engine on from scratch and the second semester, GAM350, is to put the game into the engine.

It is done using Visual Studio with Win32 API as its base and OpenGL as its Graphics Library. The importation of 3D model and animation was done with the assistant of 'Assimp' library.
